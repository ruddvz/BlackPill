# Hospital in a Box

A single-page, mobile-first reference web app combining:

- **Ayurvedic home remedies & herb notes** ("Hospital in a Box")
- **Quantum / Divine Healing Codes** — the complete set of **718 unique codes** (920 cross-referenced listings) across **25 categories**, transcribed verbatim from the source document compiled by Marc Gamma, Isabel Henn and "Reiki Doc."

## Important

This is a **spiritual / energy-healing belief practice**. It is **not medicine**, has
**no scientific proof**, and **cannot diagnose, treat, cure or prevent any disease** —
the source document says so itself. It must **never** replace professional medical or
veterinary care or prescribed treatment. In an emergency, call your local emergency
number first. A first-run safety notice must be acknowledged before the app is used.

## Features

- Complete, verbatim code set parsed directly from the original document (no paraphrasing)
- Search across every code (by number or condition) with auto-expanding sections
- Filter by category; tap any code to copy it (spaces preserved, as the source requires)
- Built-in safety flags on emergency, mental-health-crisis, animal-danger and
  "use only with medical care" codes, with crisis-line information
- Works offline — a single static `index.html`, no build step, no dependencies

## Hosting on GitHub Pages

The app is a single static file. To publish:

1. Merge this branch into your default branch.
2. Repo **Settings → Pages → Build and deployment → Source: Deploy from a branch**.
3. Choose the default branch and `/ (root)` folder, then **Save**.

The site will be served at `https://<user>.github.io/<repo>/`.
